# 쿠팡 카테고리 소싱 추첨기

쿠팡 **실제 사이트**(https://www.coupang.com)의 전체 카테고리 트리를 가져와서,
**KC 인증이 필요하거나 어린이 제품**으로 보이는 카테고리를 제외한 뒤,
소싱할 카테고리 하나(또는 여러 개)를 **랜덤으로 뽑아주는** 도구입니다.

## 동작 방식

1. 쿠팡 홈페이지를 실제 브라우저(Playwright + Chrome)로 열고, 카테고리 메뉴가 호출하는
   내부 API `n-api/web-adapter/category-list` 응답을 가로채 **전체 카테고리 트리**
   (대분류 15 / 중분류 약 250 / 소분류 약 2,400, 총 ~2,600개)를 추출합니다.
   - 쿠팡은 봇 차단(Akamai)이 있어 일반 요청은 403이 납니다. 그래서 실제 Chrome + 스텔스로 우회합니다.
   - 메뉴 DOM은 '더보기'로 잘려 있지만, 이 API에는 잘린 항목까지 전부 들어있어 누락이 없습니다.
   - 기획전(`/np/campaigns/...`) 같은 비(非)카테고리 노드는 제외합니다.
2. 카테고리 이름(대>중>소 경로) 기반 키워드로 **KC 인증 대상 / 어린이 제품**을 걸러냅니다.
   (규칙은 `filters.py`에서 자유롭게 수정 가능)
3. 남은 후보 중에서 랜덤으로 추첨합니다.

## 설치

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m playwright install chromium   # 또는 시스템 Chrome 사용(기본)
```

## 웹 UI로 사용하기 (권장)

```bash
source .venv/bin/activate
python app.py
# 브라우저에서 http://127.0.0.1:5000 접속
```

- **랜덤으로 뽑기** 버튼으로 카테고리 추첨 (분류 단계·개수 선택 가능)
- **최신 카테고리 새로고침** 버튼으로 쿠팡에서 트리를 다시 수집(10~20초, Chrome 창이 잠깐 뜸)
- 각 결과의 **쿠팡에서 보기** 링크로 해당 카테고리 페이지 이동

## CLI로 사용하기

```bash
# 1) 쿠팡에서 전체 카테고리 수집 → categories.json 생성
python main.py fetch

# 2) KC/어린이 제외 후 소분류에서 1개 랜덤 추첨
python main.py pick

# 옵션들
python main.py pick -n 5          # 5개 추첨
python main.py pick --level 2     # 중분류에서 추첨 (1=대,2=중,3=소[기본])
python main.py pick --refresh     # 추첨 전 최신 카테고리로 다시 수집
python main.py stats              # 필터 통과/제외 통계와 제외 사유 보기
```

> 수집 시 Chrome 창이 잠깐 뜹니다(차단 우회를 위해 기본 headed 모드).
> 창 없이 돌리려면 `--headless` 를 붙이세요(차단될 수 있음).

## 필터 기준 (`filters.py`)

- `EXCLUDED_L1`: 통째로 제외하는 대분류 — `출산/유아동`, `가전디지털`, `식품`, `뷰티(화장품)`, `도서/음반/DVD`
- `EXCLUDED_L2`: 특정 중분류만 제외 — `헬스/건강식품` 중 건강식품·영양제류 (운동용품은 유지)
- `CHILDREN_KEYWORDS`: 어린이/유아 제품 키워드 (유아, 아동, 완구, 기저귀 …)
- `KC_KEYWORDS`: KC 인증(전기·전자·방송통신) 키워드 (가전, 충전, 배터리, 조명 …)
- `OTHER_KEYWORDS`: 기타 제외 키워드 (성인용품, 화장품 …)

키워드 휴리스틱이라 100% 정확하진 않습니다. 소싱 후보를 1차로 거르는 용도이며,
필요에 맞게 리스트를 추가/삭제해서 기준을 조정하세요.

## 참고 / 한계

- 쿠팡 카테고리 구조가 바뀌면 `scraper.py`의 추출 로직 조정이 필요할 수 있습니다.
- 식품·화장품·성인용품·도서/음반/DVD는 요청에 따라 제외합니다.
- `헬스/건강식품` 대분류는 운동용품(홈트레이닝, 헬스/요가용품)만 남기고
  건강식품·영양제류 중분류는 제외합니다.
